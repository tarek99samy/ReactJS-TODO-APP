# TODO-APP

## A simple Todo applications for saving daily tasks and remove done ones

### Features

#### - Responsive design

#### - Can add/remove tasks

---

### Built with

#### - ReactJS

#### - parcel

#### - HTML5

#### - CSS

---

### How to run

#### - Clone the repo then type in terminal

```code
    npm install
```

#### - Then type

```code
    npm run dev
```

#### - Open your browser and follow the `http://localhost:1234`

---
